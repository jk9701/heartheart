# Interactive Heart Animation
a glowing 3D heart made of particles.  
Hold the mouse button to scatter the heart, then release it to bring it back together.

## Features
- 3D particle heart
- Smooth color-changing animation
- Hold-to-scatter interaction
- Release-to-collect animation
- Glowing neon effect
- Built with JavaScript, Three.js and GSAP

## Technologies
- HTML
- CSS
- JavaScript
- Three.js
- GSAP
